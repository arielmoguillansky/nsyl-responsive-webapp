# nsyl-responsive-webapp

A fully responsive Single Page Application (SPA) using Vue.js, Bootstrap, Vanilla JS and Firebase
<h4> Features: </h4>

  * Chat using Firebase
    * Google Auth
    * Generic mail Auth
  * Styled with Bootstrap 4
  * Reneder wiwth Vue.js
  * Google geoLoc API
  * Show Weather feature using DarkSky API
